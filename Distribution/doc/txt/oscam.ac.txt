NAME
       oscam.ac - anti-cascading table for OSCam

SYNOPSIS
       anti-cascading table

DESCRIPTIONS
       <CAID>:<provider ID>=<seconds>
	  define time cycles between CWs changes relating to CAID and provider
	  ID

       *=<seconds>
	  default time cycles between CWs changes, required

EXAMPLES
	0100:000000=10
	*=7

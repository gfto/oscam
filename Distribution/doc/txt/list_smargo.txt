NAME
       list_smargo - list all connected Smartreader+

SYNOPSIS
       list_smargo

DESCRIPTIONS
       The list_smargo software lists all connected Smartreader+ with bus number and device address.

OPTIONS
       endpoint address
	  set endpoint address,default 0x82
